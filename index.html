Welcome to SyncMLDotNet

This project includes a set of .Net components for synchronizing meta data with a SyncML server such as Funambol DS server. SyncML Client for Open Contacts and SyncML Addin for MS Outlook are utilizing the API.

End User Notice

In addition to SyncML Client API, this project also delivers  a few applications:

  1. SyncML Addin for MS Outlook 2010 - 2016
  2. SyncML Addin for MS Outlook 2007 - 2016
  3. SyncML Plugin for Open Contacts
